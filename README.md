# CSCE 431 Project 3: Load Balancer Simulation

## Overview
This project simulates a large company's load-balancing system using C++. It dynamically allocates and deallocates web servers based on the size of a request queue.

## Compilation
To compile the project, simply run the following command in the root directory:
`make`

## Execution
Run the compiled program using:
`./loadbalancer`
