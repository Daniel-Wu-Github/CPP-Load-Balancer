Detailed Description

Manages the request queue and routes jobs to an active pool of web servers.

  • The LoadBalancer handles dynamic scaling by provisioning and de-provisioning WebServers based on the current queue size. It also features a firewall mechanism to drop requests from blocked IP ranges.
