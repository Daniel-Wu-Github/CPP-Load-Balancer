\doxysubsection{Detailed Description}
Manages the request queue and routes jobs to an active pool of web servers. 


\begin{DoxyItemize}
\item The \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} handles dynamic scaling by provisioning and de-\/provisioning Web\+Servers based on the current queue size. It also features a firewall mechanism to drop requests from blocked IP ranges. 
\end{DoxyItemize}
