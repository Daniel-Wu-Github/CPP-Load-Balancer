\doxysubsection{Detailed Description}
Represents an individual processing unit in the load balancer system. 


\begin{DoxyItemize}
\item The \mbox{\hyperlink{classWebServer}{Web\+Server}} takes a single \mbox{\hyperlink{structRequest}{Request}} from the \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}}, processes it over a set number of clock cycles (time), and alerts the system when it is available for a new job. 
\end{DoxyItemize}
