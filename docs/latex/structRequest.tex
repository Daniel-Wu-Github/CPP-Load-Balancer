\doxysubsection{Detailed Description}
A struct representing a single network request. 


\begin{DoxyItemize}
\item This lower-\/level data structure encapsulates all the necessary information for a simulated network job, including source and destination IP addresses, the processing time required, and the type of job (Processing or Streaming). 
\end{DoxyItemize}
